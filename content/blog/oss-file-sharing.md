I find it more and more complicated to quickly share images or files directly between my multiple devices and operating systems over the network.

This list of OSS projects are here to help you find the right solution for you.

<!--more-->

# Localsend

https://localsend.org/

Share files to nearby devices.
Free, open-source, cross-platform.

[CLI compatible variant in Go](https://github.com/meowrain/localsend-go)

# Copyparty

https://github.com/9001/copyparty

Portable file server with accelerated resumable uploads, dedup, WebDAV, FTP, TFTP, zeroconf, media indexer, thumbnails++ all in one file, no deps

# Pairdrop

https://pairdrop.net/

PairDrop: Browser peer-to-peer Transfer Files Cross-Platform. No Setup, No Signup.

# Tailscale Taildrop

https://tailscale.com/kb/1106/taildrop

If you use tailscale, enable File Sharing in settings and easily send files across devices.

# Croc

https://github.com/schollz/croc

CLI tool to send files over internet just by using simple password. Multiplatform with unofficial [Android app](https://f-droid.org/packages/com.github.howeyc.crocgui/)
